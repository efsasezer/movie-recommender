Similarity Index Aklamas:

Kosins Benzerlii (Cosine Similarity)

Kosins benzerlii, iki vektr arasndaki ann kosinsn lerek benzerlii hesaplayan bir metriktir. Film neri sistemlerinde yaygn olarak kullanlr.

Ne e Yarar?

ki film arasndaki benzerlii ler Kullanc derecelendirme kalplarn karlatrr -1 ile 1 arasnda deer alr (1: tamamen benzer, -1: tamamen zt)

Nasl Hesaplanr?

cos() = (A B) / (||A|| ||B||)

A B: ki vektrn nokta arpm

||A|| ve ||B||: Vektrlerin byklkleri

Cosine Similarity Hangi Durumlarda Kullanlr?

neri Sistemleri: Kullanclarn gemite izledii filmlere gre benzer filmler nermek iin kullanlr.

Metin Madencilii ve NLP: Dokmanlarn veya cmlelerin benzerliini lmek iin yaygn olarak kullanlr.

Biyoinformatik: Genetik sekanslar veya protein yaplarn karlatrmada kullanlr.

Makine renmesi: Kmeleme algoritmalarnda benzer eleri gruplamak iin tercih edilir.

Film neri Sistemi Analiz Raporu

1. Teknik Aklama

Film Bazl neriler (Content-Based Filtering)

Bu teknik, filmlerin ierik zelliklerine (tr, oyuncular, ynetmen vb.) ve kullanc-film etkileim matrisine dayanarak kosins benzerlii kullanr. ki film arasndaki benzerlii hesaplamak iin kullanc derecelendirme vektrleri arasndaki ann kosinsn kullanr.

Poplerlik Bazl neriler (Popularity-Based)

Bu yaklam, filmlerin poplerlik metriklerini (ortalama puan, deerlendirme says) kullanarak neriler sunar. Minimum deerlendirme says eii ile kalite kontrol salanr.

Karma neriler (Hybrid)

ki yaklam birletirerek hem benzerlik bazl hem de poplerlik bazl nerileri dengeler. Bu sayede daha kapsaml ve eitli neriler sunulur.

import pandas as pd
import numpy as np
import warnings
warnings.filterwarnings('ignore')

class SimpleMovieRecommender:
    def __init__(self, movies_df, ratings_df):
        self.movies_df = movies_df
        self.ratings_df = ratings_df
        self.movie_ratings = self._calculate_movie_ratings()
        
    def _calculate_movie_ratings(self):
        """Film bana ortalama puanlar hesapla"""
        return self.ratings_df.groupby('movieId').agg({
            'rating': ['count', 'mean']
        }).reset_index()
    
    def get_popular_movies(self, n=5, min_ratings=100):
        """En popler filmleri getir"""
        popular = self.movie_ratings[
            self.movie_ratings[('rating', 'count')] >= min_ratings
        ].sort_values(('rating', 'mean'), ascending=False)
        
        popular.columns = ['movieId', 'rating_count', 'rating_mean']
        return popular.head(n).merge(
            self.movies_df[['movieId', 'title', 'genres']],
            on='movieId'
        )
    
    def get_similar_movies(self, movie_id, n=5):
        """Benzer filmleri bul (basit genre elemesi kullanarak)"""
        # Hedef filmin genre'larn al
        target_movie = self.movies_df[self.movies_df['movieId'] == movie_id].iloc[0]
        target_genres = set(target_movie['genres'].split('|'))
        
        # Dier filmlerin genre benzerliini hesapla
        def calculate_similarity(row):
            movie_genres = set(row['genres'].split('|'))
            return len(target_genres.intersection(movie_genres)) / len(target_genres.union(movie_genres))
        
        similar_movies = self.movies_df[self.movies_df['movieId'] != movie_id].copy()
        similar_movies['similarity'] = similar_movies.apply(calculate_similarity, axis=1)
        
        return similar_movies.nlargest(n, 'similarity')[['movieId', 'title', 'genres', 'similarity']]
    
    def get_user_recommendations(self, user_id, n=5):
        """Kullancya film nerisi yap"""
        # Kullancnn izledii filmlerin genre'larn topla
        user_movies = self.ratings_df[self.ratings_df['userId'] == user_id]
        user_movies = user_movies.merge(self.movies_df[['movieId', 'genres']], on='movieId')
        
        # Kullancnn yksek puan verdii filmler (4+ yldz)
        liked_movies = user_movies[user_movies['rating'] >= 4]
        
        if len(liked_movies) == 0:
            return self.get_popular_movies(n)
        
        # Sevilen filmlerin genre'larn say
        genre_counts = {}
        for _, row in liked_movies.iterrows():
            for genre in row['genres'].split('|'):
                genre_counts[genre] = genre_counts.get(genre, 0) + 1
        
        # zlenmemi filmleri bu genre'lara gre puanla
        unwatched = self.movies_df[~self.movies_df['movieId'].isin(user_movies['movieId'])]
        
        def score_movie(row):
            score = 0
            for genre in row['genres'].split('|'):
                score += genre_counts.get(genre, 0)
            return score
        
        unwatched['recommendation_score'] = unwatched.apply(score_movie, axis=1)
        
        return unwatched.nlargest(n, 'recommendation_score')[['movieId', 'title', 'genres', 'recommendation_score']]

def main():
    # Veri setlerini ykle
    movies_df = pd.read_csv('./movie.csv')
    ratings_df = pd.read_csv('./rating.csv')
    
    # neri sistemini olutur
    recommender = SimpleMovieRecommender(movies_df, ratings_df)
    
    # Popler filmleri gster
    print("\nPopler Filmler:")
    print(recommender.get_popular_movies())
    
    # Benzer filmleri gster
    print("\nIron Giant'a benzer filmler:")
    print(recommender.get_similar_movies(2761))
    
    # Kullanc nerilerini gster
    print("\nKullanc 1 iin neriler:")
    print(recommender.get_user_recommendations(1))

if __name__ == "__main__":
    main()

import pandas as pd
import numpy as np
from sklearn.model_selection import train_test_split

class SimpleRecommenderEvaluation:
    def __init__(self, movies_df, ratings_df):
        self.movies_df = movies_df
        self.ratings_df = ratings_df
        
    def evaluate_model(self, test_size=0.2, min_ratings=100):
        """Basit bir deerlendirme gerekletir"""
        # Popler filmleri filtrele
        movie_counts = self.ratings_df['movieId'].value_counts()
        popular_movies = movie_counts[movie_counts >= min_ratings].index
        filtered_ratings = self.ratings_df[self.ratings_df['movieId'].isin(popular_movies)]
        
        # Veriyi bl
        train_data, test_data = train_test_split(filtered_ratings, test_size=test_size, random_state=42)
        
        # Film ortalamalarn hesapla
        movie_means = train_data.groupby('movieId')['rating'].mean()
        
        # Test verisinde tahmin yap
        test_predictions = []
        test_actuals = []
        
        for _, row in test_data.iterrows():
            movie_id = row['movieId']
            actual_rating = row['rating']
            
            # Tahmin: film ortalama puan
            predicted_rating = movie_means.get(movie_id, 3.5)  # Varsaylan 3.5
            
            test_predictions.append(predicted_rating)
            test_actuals.append(actual_rating)
        
        # Metrikleri hesapla
        test_predictions = np.array(test_predictions)
        test_actuals = np.array(test_actuals)
        
        mae = np.mean(np.abs(test_predictions - test_actuals))
        rmse = np.sqrt(np.mean((test_predictions - test_actuals) ** 2))
        accuracy_1 = np.mean(np.abs(test_predictions - test_actuals) <= 1.0)
        
        return {
            'MAE': mae,
            'RMSE': rmse,
            'Accuracy_1.0': accuracy_1,
            'Test_Size': len(test_actuals)
        }

    print("Model deerlendiriliyor...")
    evaluator = SimpleRecommenderEvaluation(movies_df, ratings_df)

if __name__ == "__main__":
